# Credit Tracing Service

This service provides you to create, modify, delete and query credit applications.


## Docker Compose
```
    docker-compose up
```

## Build
```
./gradlew clean build
```

## Unit Test
```
./gradlew clean test
```


## Package
```
./gradlew clean bootJar
```

## Docker Build
```
    docker build . -t notification-service
```
